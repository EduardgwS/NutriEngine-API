API REST do aplicativo nutricional NutriEngine, construída com FastAPI e integrada ao Google Gemini para análise nutricional com IA.

## Funcionalidades

- Autenticação via Google OAuth 2.0 (Android) com JWT
- Megumi, assistente nutricional com IA (Gemini), memória de histórico e suporte a imagens
- Busca de alimentos na tabela TACO brasileira com similaridade fonética
- Identificação de alimento por foto via Gemini Vision
- Receita do dia rotativa por objetivo nutricional
- Mercado, listagem de parceiros e recomendação de produtos por perfil nutricional
